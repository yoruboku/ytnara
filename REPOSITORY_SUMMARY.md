## 🚀 **How to Use Right Now**

### Quick Start (Recommended)
```bash
# Clone the repository
git clone <your-repo-url>
cd yt-nara

# One-command setup
python3 quick_start.py --setup
```

### Test First (Demo Mode)
```bash

# If demo passes, proceed with setup
python3 yt_nara.py --setup
```

